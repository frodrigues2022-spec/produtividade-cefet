# Guia de Produtividade: Organizando sua vida com o Notion

Bem-vindos ao guia do Grupo 05 para o projeto **Conecta BSI**. O nosso objetivo é ajudar você a organizar seus estudos e tarefas diárias de forma simples e eficiente.

## O que é o Notion?
O Notion é uma ferramenta incrível que funciona como um "espaço de trabalho tudo-em-um". Você pode criar listas, quadros de tarefas, calendários e muito mais.

## Por que usar?
- **Centralização:** Tudo em um só lugar.
- **Visualização:** Você escolhe como quer ver suas tarefas (lista ou quadro).
- **Simplicidade:** Fácil de começar, mesmo sem experiência.

## Como começar (Passo a Passo)
1. **Crie sua conta:** Acesse [notion.so](https://www.notion.so) e cadastre-se.
2. **Use nosso Template:** Nós criamos um quadro de tarefas pronto para você! [Clique aqui para acessar nosso Template de Organização]. (Substitua este texto pelo link do seu template público).
3. **Personalize:** Arraste suas tarefas entre as colunas "A Fazer", "Fazendo" e "Concluído".
